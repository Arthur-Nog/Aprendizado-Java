Arrays:
-

Sintaxe:  

tipo [ ] nome_doArray = new int [espaço_em_memória];   

-> Lado esquerdo é do tipo reference. Direito é um objeto.  
-> Arrays são objetos na memória.

-> Faz referência a um lugar na memória.  

-> Se não tiver nenhum valor definido, o Array possui um valor padrão.   

VALORES PADRÃO NO ARRAY:

    -byte, short, int, long, float, double -> Valor padrão 0.
    -char -> '\u0000'
    -boolean -> false
    - String -> null
